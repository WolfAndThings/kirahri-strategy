# Master Project List
## Everything You Need to Create — Organized by Batch

---

## HOW THIS WORKS

Pick a project. Film everything in that project in one session. Move to the next project.

Each project lists:
- **VISUALS NEEDED** — Every shot/photo you need
- **SCRIPTS** — What to say (word-for-word)
- **CAPTIONS** — Ready to copy-paste
- **CHECKLIST** — Your to-do list
